# Kanishk Patil - Developer Portfolio

## Overview
A modern, responsive developer portfolio website for Kanishk Patil, a First Year B.E. Computer Engineering student at FR.CRCE, Mumbai.

## Project Structure
```
/
├── public/               # Static assets served by Express
│   ├── index.html        # Main HTML file
│   ├── styles.css        # CSS with modern styling
│   ├── script.js         # JavaScript for interactivity
│   └── images/           # Profile and gallery images
├── server.js             # Express server (port 5000)
├── package.json          # Node.js dependencies
└── .gitignore            # Git ignore rules
```

## Features
- Responsive navigation with mobile menu toggle
- Hero section with profile image and featured project
- About section with education, skills, and goals
- Projects grid with placeholder links
- Interactive photo gallery with lightbox
- Blog section with article cards
- Contact form with validation
- Smooth scroll navigation
- Glass-morphism design effects
- Mobile-responsive layout

## Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- Express.js for serving static files
- Google Fonts (Inter)

## Running the Project
The server runs on port 5000 and serves the static files from the public folder.
