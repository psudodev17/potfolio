# Kanishk Portfolio Website

## Overview
A modern, responsive portfolio website for Kanishk Patil built with vanilla HTML, CSS, and JavaScript. Features a glass-morphism design with smooth animations and a clean, professional look.

## Project Structure
```
/
├── index.html      # Main HTML file with all sections
├── styles.css      # CSS styles with glass-morphism design
├── script.js       # JavaScript for interactions and animations
├── server.js       # Node.js static file server
└── replit.md       # Project documentation
```

## Features
- **Hero Section**: Profile image, tagline, CTA buttons, featured project highlight
- **About Section**: Education, experience, skills, achievements, and interests
- **Projects Grid**: Showcase of selected projects with images and links
- **Services Section**: Web development, portfolio building, consulting offerings
- **Gallery**: Image grid with hover effects
- **Blog Section**: Blog post previews with categories
- **Contact Form**: Working form with JavaScript validation
- **Responsive Design**: Fully responsive across all device sizes
- **Smooth Scrolling**: Smooth navigation between sections
- **Mobile Menu**: Hamburger menu for mobile devices
- **Scroll Animations**: Fade-in animations on scroll using Intersection Observer

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, backdrop-filter, custom properties)
- Vanilla JavaScript (ES6+)
- Node.js (static file server)

## Running the Project
The server runs on port 5000:
```bash
node server.js
```

## Customization
- Replace profile image URL in `index.html`
- Update project images and links in the projects section
- Modify contact email and social media links
- Add real resume file for download functionality
- Update blog posts with actual content

## Design System
- **Colors**: Sky-600 to Indigo-600 gradient for primary actions
- **Typography**: Inter font family
- **Glass Effect**: rgba backgrounds with backdrop-filter blur
- **Shadows**: Layered shadows for depth (sm, md, lg, xl)
- **Transitions**: 0.3s ease for smooth interactions
